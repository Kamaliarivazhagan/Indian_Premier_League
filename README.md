# Indian Premier League Database

This project is a SQL database based on the Indian Premier League (IPL).

## Description
The database contains tables for:
- Teams
- Players
- Matches
- Seasons
- Player_Statistics
- Awards
- stadium
- team_captains
- toss_details
- umpire

## Purpose
This project demonstrates database design, relationships, and SQL queries used to analyze IPL data such as player performance, team results, and match statistics.

## Technologies Used
- MySQL
- SQL
- Database
- Data Analytics
